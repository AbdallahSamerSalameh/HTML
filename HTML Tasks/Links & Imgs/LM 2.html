What is the diffrence between HTML and CSS

1)HTML (HyperText Markup Language) is the standard markup language for creating web pages and web applications. It provides the structure of a webpage, while CSS (Cascading Style Sheets) is used to control the presentation, formatting, and layout.

4)CSS uses a set of rules, such as selectors, properties, and values, to apply style to the elements defined in the HTML document.
